# System Requirements

To run the CLI you need the following:

| Tool | Minimum Version | Notes |
| ---- | --------------- | ----- |
| **Node.js** | 18.x | Includes `npm` package manager |
| **Docker** | 27.x | Linux or Docker Desktop; enable *VirtioFS* on macOS for faster volumes |
| **Kurtosis** | Latest | Follow the [official installer](https://docs.kurtosis.com/install) |

<Callout type="warning">On Windows, use **WSL 2** or run inside a Linux VM for the best experience.</Callout>
