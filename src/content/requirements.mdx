# System Requirements

To run the CLI you need the following:

| Tool | Minimum Version | Notes |
| ---- | --------------- | ----- |
| [**Node.js**](https://nodejs.org/en/download) | 18.x | Includes `npm` package manager |
| [**Docker**](https://docs.docker.com/get-started/get-docker/) | 27.x | Linux or Docker Desktop; enable *VirtioFS* on macOS for faster volumes |
| [**Kurtosis**](https://docs.kurtosis.com/install) | Latest | Follow the [official installer](https://docs.kurtosis.com/install) |
