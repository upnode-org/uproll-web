# CLI Menu Reference

Run the menu at any time:

```bash
uproll run
```

![cli menu](/docs/cli-menu.gif)


Below is a quick index of the menu items. Click into each page to learn more.

| Command | Purpose |
| ------- | ------- |
| **Deploy Rollup** | Deploys a new rollup |
| **Stop Rollup** | Stop and delete an active rollup |
| **Chain Info** | View generated artifacts (genesis, walletsâ€¦) |
| **Status ...** | Show deployment health |
| **View Logs** | Tail logs for rollup services |
| **Clean** | Remove **all** Docker containers & project folders | 
| **Exit** | Quits and exits the cli focus, `uproll run` can be ran again to enter the menu again |
