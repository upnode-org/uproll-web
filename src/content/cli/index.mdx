# CLI Command Reference

Run the menu at any time:

```bash
uproll run
```

Below is a quick index of sub‑commands. Click into each page to learn more.

| Command | Purpose |
| ------- | ------- |
| **stop‑rollup** | Stop and delete an active rollup |
| **chain‑info** | View generated artifacts (genesis, wallets…) |
| **status** | Show deployment health |
| **view‑logs** | Tail logs for rollup services |
| **clean** | Remove **all** Docker containers & project folders |
