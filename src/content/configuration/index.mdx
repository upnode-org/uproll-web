# Configuration Philosophy

UpRoll splits configuration into **logical domains** (wallet, layer 1, signer, rollup, gas, data availability). Each CLI prompt maps 1‑to‑1 with the tables linked in the sidebar, so you can look up a single field without wading through a giant page.

<Callout type="tip">Prefer copy‑paste? Every table cell is selectable – double‑click to copy default values.</Callout>
