# Signer Configuration

| Parameter | Description |
| --------- | ----------- |
| **Batcher** | Private key *or* signer endpoint for the batcher |
| **Sequencer** | Private key *or* signer endpoint for the sequencer |
| **Proposer** | Private key *or* signer endpoint for the proposer |
| **Challenger** | Private key *or* signer endpoint for the challenger |

<Callout type="info">Need a signer endpoint? Check out the open‑source <a href="https://github.com/upnodedev/signer-proxy">signer‑proxy</a> tool.</Callout>
