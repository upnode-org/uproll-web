# Signer Configuration

| Parameter | Description |
| --------- | ----------- |
| **Batcher** | Private key *or* signer endpoint for the batcher |
| **Sequencer** | Private key *or* signer endpoint for the sequencer |
| **Proposer** | Private key *or* signer endpoint for the proposer |
| **Challenger** | Private key *or* signer endpoint for the challenger |

<Callout type="info">Need a signer endpoint? We provide a [signer proxy tool](https://github.com/upnodedev/signer-proxy) that can be used as a signer endpoint to sign transactions for any of these roles. Instructions for setting up a signer address are also given there.</Callout>
