# Gas Configuration

| Parameter | Description |
| --------- | ----------- |
| **Block Gas Limit** | Max gas per block |
| **EIP‑1559 Elasticity** | Elasticity multiplier |
| **EIP‑1559 Denominator** | Denominator for the fee model |
| **Base Fee Scalar** | Multiplier for base fee |
| **Blob Base Fee Scalar** | Multiplier for blob base fee |
