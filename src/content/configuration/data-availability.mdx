# Data Availability Configuration

| Parameter | Description |
| --------- | ----------- |
| **DA Type** | `auto`, `blobs`, `calldata`, or `custom` |
| **Batcher Submission Frequency (min)** | Interval between batch submissions |
| **DA Server Endpoint** | URL when **custom** DA type |
| **Commitment Type** | `Generic` or `KeccakCommitment` |
| **Challenge Contract Address** | If commitment type is **Generic** |
| **Challenge Window** | Duration for challenges |
| **Resolve Window** | Duration to resolve disputes |
| **Bond Size** | Stake required for disputes |
| **Refund %** | Refund percentage after resolution |
