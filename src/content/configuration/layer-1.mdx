# Layer 1 Configuration

| Parameter | Description |
| --------- | ----------- |
| **Select Settlement Layer** | ETH Mainnet, ETH Sepolia, or **Custom** |
| **Enter the Custom Chain ID** | Chain ID for a custom L1 |
| **Enter the EL RPC URL** | Execution‑layer RPC |
| **Enter the EL WS URL** | WebSocket endpoint for EL |
| **Enter the CL RPC URL** | Consensus‑layer RPC |
| **Select RPC Kind for L1** | Auto‑detected (Alchemy, QuickNode or Standard) |
