# Deployment Workflows

UpRoll offers **two paths** to a running OP Stack chain:

| Workflow | Ideal when… | UX |
| -------- | ----------- | -- |
| **Website + CLI** | You prefer a graphical UI for complex configs | Web wizard → `uproll deploy -i <id>` |
| **CLI only** | You live in the terminal or need CI automation | `uproll run` interactive wizard |
