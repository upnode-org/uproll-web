# Quick Start ⏱️

Follow these **five commands** to go from zero to a local OP Stack chain:

```bash
# 1 Install
npm install -g uproll-cli

# 2 Create & deploy in one go (CLI wizard)
uproll run

# 3 — or — Use the website & supply the config‑ID
uproll deploy -i <config_id>

# 4 Check status
uproll status

# 5 Stop and cleanup when finished
uproll clean
```

You can now query the L2 RPC at `http://localhost:8545`.
